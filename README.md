# OfferHunter
## 软件需求工程大作业—微信小程序

该项目前端界面使用的是微信小程序，使用了vant的UI组件库和wxcharts组件,后端实现了job搜索和猎聘网爬取功能，采用flask框架，部署在阿里云服务器上
